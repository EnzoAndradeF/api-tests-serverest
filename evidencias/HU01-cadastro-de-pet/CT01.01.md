**CT01 – Cadastro de pet com dados válidos**

Request:
- Método: POST
- Endpoint: /pet
- Body enviado:
```json
{
  "id": 10,
  "name": "Rex",
  "status": "available"
}
````

- Status Code: 200

````json
{
  "id": 10,

  "status": "available"
}
````
