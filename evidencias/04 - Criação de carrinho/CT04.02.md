**CT04.02 – Adicionar quantidade de produto maior que estoque no carrinho.**

Request:
- Método: POST
- Endpoint: /produtos
- Body enviado:
```json
{
  "produtos": [
    {
      "idProduto": "PYmReWtX1ijiRY9u",
      "quantidade": 101
    }
  ]
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "message": "Produto não possui quantidade suficiente",
    "item": {
        "idProduto": "PYmReWtX1ijiRY9u",
        "quantidade": 101,
        "quantidadeEstoque": 100,
        "index": 0
    }
}
````
Resultado: Passou
