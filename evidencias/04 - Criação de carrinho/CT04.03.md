**CT04.03 – Adicionar produto inexistente no carrinho.**

Request:
- Método: POST
- Endpoint: /produtos
- Body enviado:
```json
{
  "produtos": [
    {
      "idProduto": "inexistente",
      "quantidade": 1
    }
  ]
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "message": "Produto não encontrado",
    "item": {
        "idProduto": "inexistente",
        "quantidade": 1,
        "index": 0
    }
}
````
Resultado: Passou
