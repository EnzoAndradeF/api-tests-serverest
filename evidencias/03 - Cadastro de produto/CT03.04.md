**CT03.04 – Realizar o cadastro de produto com quantidade negativa**

Request:
- Método: POST
- Endpoint: /produtos
- Body enviado:
```json
{
    "nome": "notebook lenovo ideapad5",
    "preco": 700,
    "descricao": "Notebook",
    "quantidade": -100
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "quantidade": "quantidade deve ser maior ou igual a 0"
}
````
Resultado: Passou
