**CT03.02 – Realizar o cadastro de produto com nome existente**

Request:
- Método: POST
- Endpoint: /produtos
- Body enviado:
```json
{
    "nome": "notebook lenovo ideapad3",
    "preco": 700,
    "descricao": "Notebook",
    "quantidade": 100
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "message": "Já existe produto com esse nome"
}
````
Resultado: Passou
