**CT03.03 – Realizar o cadastro de produto com preço negativo**

Request:
- Método: POST
- Endpoint: /produtos
- Body enviado:
```json
{
    "nome": "notebook lenovo ideapad4",
    "preco": -700,
    "descricao": "Notebook",
    "quantidade": 100
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "preco": "preco deve ser um número positivo"
}
````
Resultado: Passou
