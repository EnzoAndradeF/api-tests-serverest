**CT03.01 – Realizar o cadastro de produto com usuário administrador**

Request:
- Método: POST
- Endpoint: /produtos
- Body enviado:
```json
{
    "nome": "notebook lenovo ideapad3",
    "preco": 700,
    "descricao": "Notebook",
    "quantidade": 100
}
````
Responde:
- Status Code: 201
- Body retornado:
````json
{
    "message": "Cadastro realizado com sucesso",
    "_id": "PYmReWtX1ijiRY9u"
}
````
