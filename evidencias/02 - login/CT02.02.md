**CT02.02 – Realizar o login com email inexistente**

Request:
- Método: POST
- Endpoint: /login
- Body enviado:
```json
{
  "email": "emailinexistente@qa.com.br",
  "password": "teste"
}
````
Responde:
- Status Code: 401
- Body retornado:
````json
{
    "message": "Email e/ou senha inválidos"
}
````
Resultado: Passou
