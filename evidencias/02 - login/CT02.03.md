**CT02.03 – Realizar o login com campos vazios**

Request:
- Método: POST
- Endpoint: /login
- Body enviado:
```json
{
  "email": "",
  "password": ""
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "email": "email é obrigatório",
    "password": "password é obrigatório"
}
````
Resultado: Passou
