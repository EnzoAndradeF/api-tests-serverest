**CT01.06 – Cadastro com campo email já cadastrado**

Request:
- Método: POST
- Endpoint: /usuarios
- Body enviado:
```json
{
    "nome": "enzo",
    "email": "fulano@qa.com",
    "password": "teste",
    "administrador": "true"
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "message": "Este email já está sendo usado"
}
````
