**CT01.04 – Cadastro com campo "password" vazio**

- Método: POST
- Endpoint: /usuarios

```json
{
    "nome": "teste01.03",
    "email": "teste01.03@qa.com.br",
    "password": "",
    "administrador": "false"
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "password": "password não pode ficar em branco"
}
````
