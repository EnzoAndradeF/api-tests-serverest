**CT01.05 – Cadastro com campo "administrador" vazio**

Request:
- Método: POST
- Endpoint: /usuarios
- Body enviado:
```json
{
    "nome": "teste01.04",
    "email": "teste01.04@qa.com.br",
    "password": "teste",
    "administrador": ""
}
````
Responde:
- Status Code: 400
- Body retornado:
````json
{
    "administrador": "administrador deve ser 'true' ou 'false'"
}
````
Resultado: Passou
